# vacation_ai

Agente de viagem que recomenda hóteis e passeios, além de informar sobre o clima no período escolhido para viajar.

![VacationAI](https://symmetrical-space-zebra-7vrrwxpx756r3rvqx-8000.app.github.dev/)

## 🚀 Demo
Digite qualquer destino e receba:
- Clima em tempo real
- Melhores hotéis por região
- Top atrações com justificativa
- O que levar na mala
- Dicas de transporte e moeda

## 🎨 Tech Stack

**Backend**
- FastAPI
- Python 3.11+
- Groq API - Llama 3.3 70B
- Open-Meteo - Clima
- Nominatim - Geocoding

**Frontend**
- HTML5 + CSS3 + Vanilla JS
- Fonte Montserrat
- Design dark com gradientes rosa/roxo/azul
- 100% Responsivo
